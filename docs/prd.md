# 需求文档

## 1. 应用概述

**应用名称**：蓝牙数据监测预警

**应用描述**：一款基于React Native + Expo SDK 55开发的移动端应用，使用react-native-ble-plx实现真实蓝牙低功耗（BLE）通信，用于扫描、连接蓝牙设备，实时监测数据收发情况，并提供信号强度、数据异常、连接状态的预警功能，帮助用户实时掌握蓝牙设备通信状态。新增蓝牙图传和热相分析功能，支持接收设备端图像数据并进行热成像可视化分析。新增RGB文件与图像文件导入及分析功能，支持本地文件导入、RGB原始数据解析、图像质量分析等功能。新增机器学习模型比对分析功能，通过纯TypeScript/JS实现参考图像特征模型构建与实时蓝牙图传数据流比对分析，输出相似度评分和异常区域检测。

**技术说明**：
- 使用react-native-ble-plx实现真实BLE扫描、连接、读写、通知订阅
- 需要expo-dev-client支持（Expo Go不支持原生BLE）
- 使用expo-document-picker实现文件选择器功能
- 纯TypeScript/JS实现特征提取和比对算法，不引入TensorFlow.js等重型ML框架

## 2. 用户与使用场景

**目标用户**：需要监测蓝牙设备通信状态的技术人员、设备调试人员、物联网开发者、热成像设备使用者、图像质量分析人员、图像比对分析人员

**核心使用场景**：
- 蓝牙设备调试与测试
- 蓝牙通信质量监测
- 设备连接状态实时监控
- 数据收发异常排查
- 蓝牙图像数据接收与查看
- 热成像数据实时分析与温度监测
- RGB原始文件导入与解析
- 图像质量分析与评估
- 参考图像模型构建与实时图像流比对分析
- 图像相似度评分与异常区域检测

## 3. 页面结构与功能说明

### 页面结构

```
蓝牙数据监测预警APP
├── 设备扫描页
├── 数据监测页
├── 预警记录页
├── 数据统计页
├── 蓝牙图传页
├── 热相分析页
├── 文件分析页
├── 模型比对页
└── 设置页
```

### 3.1 设备扫描页

**功能说明**：
- 调用真实BLE API扫描周围蓝牙设备，展示设备列表
- 设备列表显示：设备名称、MAC地址、信号强度RSSI
- 点击设备进行连接操作
- 显示当前连接状态（未连接/连接中/已连接）
- 已连接设备支持断开操作
- 连接成功后自动发现Services和Characteristics
- 连接成功后自动跳转至数据监测页

### 3.2 数据监测页

**功能说明**：
- 显示当前连接设备信息（名称、MAC地址、RSSI）
- 通过BLE通知（Notify）特征值订阅实时接收数据（RX），数据格式为Base64解码后的字节流
- 通过BLE写入（WriteWithoutResponse或Write）特征值发送数据（TX）
- 数据显示支持十六进制和ASCII两种格式切换
- 数据收发历史日志记录，按时间倒序展示
- 提供数据发送输入框，支持手动发送数据到已连接设备，支持HEX/ASCII输入格式
- 显示实时数据流速率（bytes/s）
- 断开连接按钮
- **解析视图**：实时展示解析后的各字段最新值（字段名、数值、单位）
- 解析视图支持显示用户自定义的字段标签和单位

### 3.3 预警记录页

**功能说明**：
- 展示预警历史记录列表
- 预警记录包含：预警类型、触发时间、设备信息、预警详情
- 预警类型包括：信号强度预警、数据异常预警、连接断开预警、数据内容报警、相似度骤降告警
- 支持按时间倒序查看

### 3.4 数据统计页

**功能说明**：
- 显示当前连接设备的统计信息
- RSSI信号强度历史折线图，展示信号强度变化趋势
- 收发数据量统计（总接收字节数、总发送字节数）
- 实时数据流速率统计（bytes/s）
- 各解析字段历史趋势折线图，展示各字段数值随时间变化趋势

### 3.5 蓝牙图传页

**功能说明**：
- 通过BLE通知订阅接收设备端分包发送的图像数据
- 接收数据包含包序号、总包数、数据内容（JPEG或RAW格式）
- 实时显示接收进度（已接收包数/总包数）
- 接收完成后自动合并还原为完整图片并显示
- 支持保存当前图片到相册
- 历史图片列表展示，按接收时间倒序排列
- 点击历史图片可查看大图
- 支持删除历史图片

### 3.6 热相分析页

**功能说明**：
- 通过BLE通知订阅接收热成像数据（温度矩阵或热图像帧）
- 实时显示热成像可视化图像，支持伪彩色映射（色温图）
- 显示当前帧的最高温度、最低温度、平均温度及对应位置标注
- 支持触摸屏幕框选区域，显示框选区域内的最高/最低/平均温度
- 历史帧列表展示，按接收时间倒序排列
- 支持回放历史帧，查看历史热成像数据
- 支持保存当前热成像图像到相册

### 3.7 文件分析页

**功能说明**：
- 文件导入按钮，使用expo-document-picker从文件选择器选取文件
- 支持导入文件类型：RGB原始文件（.rgb / .raw格式）、JPG/JPEG、PNG图像文件
- RGB文件导入时需输入参数：宽度、高度、通道数（RGB=3 / RGBA=4 / 灰度=1）、像素深度（uint8/uint16）
- 图像文件导入后自动解析并显示
- 图像显示区域：展示解析后的图像
- 通道分离查看：支持切换显示R/G/B单通道伪彩色图像
- 像素级信息查看：点击或长按图像显示该像素的RGB值
- 图像分析结果展示：
  - 基础统计：各通道均值、方差、最大值、最小值
  - 直方图：各通道分256级显示柱状图
  - 亮度分析：计算亮度分布，标记过曝区域（>250）和欠曝区域（<5）
  - 颜色分布：RGB三通道直方图叠加显示
  - 对比度分析：显示RMS对比度值
  - 图像质量评分：综合亮度、对比度、噪声给出0-100分
- 导入历史列表：展示最近20个文件，显示文件名、大小、导入时间、分辨率等元数据
- 支持删除历史记录
- UI风格：工业级暗色仪表盘风格（#121212背景、#00E5FF电光蓝、等宽字体、2px圆角）

### 3.8 模型比对页

**功能说明**：

#### 3.8.1 参考模型构建
- 支持从文件分析页已导入的图像选择作为参考模型，或重新导入图像
- 仅支持RGB/RAW格式图像进行完整特征提取，JPG/PNG格式显示提示说明
- 特征提取包含：
  - 颜色直方图特征（RGB三通道各256级）
  - 统计矩特征（均值、方差、偏度、峰度，每通道4个值）
  - 纹理特征：局部二值模式（LBP）直方图（256级）
  - 空间网格特征：图像划分为4×4=16个子块，每块提取均值和方差（3通道×16块×2=96维向量）
- 显示参考模型信息：参考图像缩略图、特征维度数、构建时间

#### 3.8.2 实时比对分析
- 数据源：BLE图传页面接收到的最新图像帧（自动检测新帧并比对）或手动从文件导入测试图像
- 比对算法：
  - 直方图交叉相似度（Histogram Intersection，范围0~1）
  - 余弦相似度（Cosine Similarity，基于特征向量，范围-1~1）
  - 综合相似度评分（加权平均，范围0~100分）
- 差异可视化：像素级差值图（|A - B|），以伪彩色热图展示差异强度
- 异常区域检测：基于4×4子块的均值差异，标注差异超过阈值的子块（用红色矩形框标注）
- 实时模式：BLE图传收到新帧时自动触发比对，显示最新比对结果
- 手动模式：手动选择测试图像与参考模型比对

#### 3.8.3 比对结果展示
- 综合相似度评分（0~100，显示为大数字+颜色编码）
- 各算法分项评分（直方图相似度、余弦相似度）
- 差异热图展示（参考图、测试图、差异图三联对比显示）
- 异常子块标注（4×4网格中，超过差异阈值的子块高亮显示）
- 比对历史记录（最近20条，每条记录：时间、评分、测试图缩略图）
- 差异阈值可调节（滑块或数字输入，默认20%）

#### 3.8.4 实时数据流监控面板
- 显示最近10帧的相似度评分趋势折线图
- 相似度骤降告警（评分低于阈值自动触发预警，记录到预警系统）
- 平均相似度、最低相似度、最高相似度统计

#### 3.8.5 UI风格
- 工业级暗色仪表盘风格（#121212背景、#00E5FF电光蓝、等宽字体、2px圆角）

### 3.9 设置页

**功能说明**：
- 预警阈值配置：
  - RSSI信号强度阈值设置（低于该值触发预警）
  - 数据超时时长设置（超过该时长未收到数据触发预警）
  - 数据内容报警阈值设置：支持添加、编辑、删除报警规则，每条规则包含字段名、比较运算符（>、<、>=、<=、==）、阈值数值
  - 相似度骤降阈值设置（低于该值触发预警）
- 数据显示格式设置：默认显示格式选择（十六进制/ASCII）
- 日志保存设置：是否自动保存数据收发日志
- 字段映射配置：自定义字段名对应的显示标签和单位（如：T → 温度，单位℃）
- **BLE配置项**：
  - Service UUID（可自定义）
  - RX特征值UUID
  - TX特征值UUID
  - 提供常用协议默认值（如Nordic UART Service）
- **图传配置**：
  - 图像数据特征值UUID
  - 图片保存路径设置
- **热相配置**：
  - 热成像数据特征值UUID
  - 伪彩色映射方案选择（铁红、彩虹、灰度等）
  - 温度单位设置（摄氏度/华氏度）
- **模型比对配置**：
  - 差异阈值默认值设置（默认20%）
  - 相似度骤降告警阈值设置

## 4. 业务规则与逻辑

### 4.1 蓝牙连接规则
- 同一时间仅支持连接一个蓝牙设备
- 连接新设备前需先断开当前连接
- 连接过程中禁止重复连接操作
- 连接后自动发现Services和Characteristics
- 根据设置页配置的Service UUID和Characteristic UUID进行读写和通知订阅

### 4.2 数据收发规则
- 仅在设备已连接状态下可发送数据
- 数据接收（RX）：通过BLE通知（Notify）特征值订阅实时接收数据，数据格式为Base64解码后的字节流
- 数据发送（TX）：通过BLE写入（WriteWithoutResponse或Write）特征值发送数据，支持HEX/ASCII输入格式
- 数据收发日志实时记录，包含时间戳、数据内容、数据方向（RX/TX）
- 数据流速率每秒更新一次

### 4.3 数据分组解析规则
- 支持解析格式：`字段名:数值`，多组用逗号分隔（如：`T:25.3,D:1024,S:60`）
- 解析后提取字段名和数值，根据字段映射配置显示对应标签和单位
- 解析失败或格式不匹配时，不影响原始数据显示
- 解析后的字段值实时更新至解析视图
- 解析后的字段值记录历史数据，用于统计页趋势图展示

### 4.4 预警触发规则
- **信号强度预警**：当RSSI低于设置阈值时触发预警
- **数据异常预警**：当超过设置的超时时长未收到数据时触发预警
- **连接断开预警**：当设备意外断开连接时触发预警
- **数据内容报警**：当解析后的字段值满足已配置的报警规则时触发预警（如：T值 > 80）
- **相似度骤降告警**：当模型比对页实时比对评分低于设置阈值时触发预警
- 预警触发后记录至预警历史

### 4.5 数据格式转换规则
- 十六进制格式：以空格分隔的两位十六进制数（如：4A 5F 3C）
- ASCII格式：直接显示可打印字符，不可打印字符显示为点号（.）

### 4.6 权限处理规则
- Android：运行时请求BLUETOOTH_SCAN、BLUETOOTH_CONNECT、ACCESS_FINE_LOCATION权限
- iOS：需要NSBluetoothAlwaysUsageDescription权限说明
- 权限拒绝时提示用户授予权限

### 4.7 连接状态管理规则
- 真实断开检测：监听BLE连接状态变化
- 重连逻辑：连接意外断开时触发预警，用户可手动重连
- 错误处理：权限拒绝、设备不可达等场景提供友好提示

### 4.8 蓝牙图传规则
- 通过BLE通知订阅接收图像数据包，每个数据包包含包序号、总包数、数据内容
- 数据包格式：前2字节为包序号，第3-4字节为总包数，后续为图像数据内容
- 按包序号顺序缓存接收到的数据包
- 接收完所有数据包后，按序号合并为完整图像数据
- 支持JPEG和RAW格式图像数据解析
- 图像合并完成后自动显示，并保存至历史记录
- 历史图片仅保留最近50张

### 4.9 热相分析规则
- 通过BLE通知订阅接收热成像数据，数据格式为温度矩阵或热图像帧
- 温度矩阵格式：前4字节为宽度和高度（各2字节），后续为温度数值数组（每个温度值2字节，单位0.1℃）
- 根据温度数值范围和伪彩色映射方案生成色温图
- 实时计算并显示当前帧的最高温度、最低温度、平均温度
- 框选区域温度分析：计算框选区域内所有像素点的最高/最低/平均温度
- 历史帧仅保留最近100帧
- 温度单位根据设置页配置进行转换（摄氏度/华氏度）

### 4.10 文件导入与解析规则
- 使用expo-document-picker从文件选择器选取文件
- 支持文件类型：.rgb、.raw、.jpg、.jpeg、.png
- RGB原始文件导入时必须输入宽度、高度、通道数、像素深度参数
- 根据输入参数将原始字节流解析为图像数据
- 图像文件（JPG/PNG）导入后自动解析宽高和通道数
- 解析成功后渲染显示图像

### 4.11 图像分析规则
- 基础统计：计算各通道（R/G/B）的均值、方差、最大值、最小值
- 直方图：统计各通道0-255每个灰度级的像素数量，生成256级柱状图
- 亮度分析：计算亮度值（0.299R + 0.587G + 0.114B），统计亮度分布，标记过曝区域（亮度>250）和欠曝区域（亮度<5）
- 颜色分布：将RGB三通道直方图叠加显示
- 对比度分析：计算RMS对比度值（像素值标准差）
- 图像质量评分：综合亮度分布、对比度、噪声估计给出0-100分评分

### 4.12 文件管理规则
- 导入历史列表仅保留最近20个文件
- 历史记录包含文件名、文件大小、导入时间、分辨率（宽x高）、通道数等元数据
- 支持删除单个历史记录
- 删除历史记录不影响已导入的图像数据

### 4.13 参考模型构建规则
- 仅支持RGB/RAW格式图像进行完整特征提取
- JPG/PNG格式图像显示提示说明，无法进行完整特征提取
- 特征提取包含：
  - 颜色直方图特征：RGB三通道各256级，共768维
  - 统计矩特征：均值、方差、偏度、峰度，每通道4个值，共12维
  - 纹理特征：局部二值模式（LBP）直方图，256级，共256维
  - 空间网格特征：4×4=16个子块，每块提取均值和方差，3通道×16块×2=96维
- 特征提取完成后保存为参考模型，记录构建时间和特征维度数

### 4.14 实时比对分析规则
- 数据源：BLE图传页面接收到的最新图像帧或手动导入的测试图像
- 实时模式：BLE图传收到新帧时自动触发比对
- 手动模式：用户手动选择测试图像与参考模型比对
- 比对算法：
  - 直方图交叉相似度：计算参考图与测试图直方图的交叉值，范围0~1
  - 余弦相似度：计算参考图与测试图特征向量的余弦相似度，范围-1~1
  - 综合相似度评分：加权平均直方图相似度和余弦相似度，归一化为0~100分
- 差异可视化：计算像素级差值图（|A - B|），以伪彩色热图展示差异强度
- 异常区域检测：基于4×4子块的均值差异，标注差异超过阈值的子块（用红色矩形框标注）
- 比对结果记录至历史记录，仅保留最近20条

### 4.15 相似度骤降告警规则
- 实时模式下，当综合相似度评分低于设置阈值时触发相似度骤降告警
- 告警记录至预警记录页，包含触发时间、评分、测试图缩略图
- 告警阈值可在设置页配置

### 4.16 实时数据流监控规则
- 记录最近10帧的相似度评分，生成趋势折线图
- 计算平均相似度、最低相似度、最高相似度统计值
- 实时更新监控面板数据

## 5. 异常与边界情况

| 场景 | 处理方式 |
|------|----------|
| 蓝牙未开启 | 提示用户开启蓝牙功能 |
| 扫描无设备 | 显示「未发现设备」提示 |
| 连接失败 | 提示连接失败原因，返回设备扫描页 |
| 连接中断 | 触发连接断开预警，自动返回设备扫描页 |
| 发送数据失败 | 提示发送失败，保留输入内容 |
| 无权限访问蓝牙 | 提示用户授予蓝牙权限 |

## 6. 验收标准

1. 打开应用，授予蓝牙权限，进入设备扫描页，调用真实BLE API扫描到周围蓝牙设备列表
2. 点击目标设备，成功连接并自动发现Services和Characteristics，跳转至数据监测页
3. 在数据监测页通过BLE通知订阅实时查看接收到的数据（RX）和通过BLE写入发送的数据（TX）
4. 接收到包含分组前缀的数据（如：`T:25.3,D:1024`），解析视图实时显示各字段最新值
5. 进入设置页，配置数据内容报警规则（如：T > 80）和BLE配置项（Service UUID、RX/TX特征值UUID），保存成功
6. 当接收数据中T值超过80时，触发数据内容报警并记录至预警记录页
7. 进入蓝牙图传页，接收设备端分包发送的图像数据，实时显示接收进度，接收完成后自动合并还原为完整图片并显示，保存图片到相册成功
8. 进入热相分析页，接收热成像数据，实时显示伪彩色热成像图像，显示最高/最低/平均温度标注，框选区域查看区域温度分析结果
9. 进入文件分析页，点击文件导入按钮，使用expo-document-picker选择一个RGB原始文件（.rgb格式），输入宽度、高度、通道数（RGB=3）、像素深度（uint8），解析成功并显示图像，查看基础统计、直方图、亮度分析、对比度分析、图像质量评分等分析结果
10. 在文件分析页点击图像，显示该像素的RGB值，切换通道分离查看R/G/B单通道伪彩色图像
11. 进入模型比对页，从文件分析页已导入的RGB格式图像中选择一张作为参考模型，完成特征提取，显示参考模型信息（缩略图、特征维度数、构建时间）
12. 在模型比对页开启实时模式，BLE图传接收到新帧时自动触发比对，显示综合相似度评分（0~100）、各算法分项评分、差异热图（参考图、测试图、差异图三联对比）、异常子块标注（4×4网格中超过差异阈值的子块高亮显示）
13. 在模型比对页查看实时数据流监控面板，显示最近10帧的相似度评分趋势折线图、平均相似度、最低相似度、最高相似度统计
14. 当相似度评分低于设置阈值时，触发相似度骤降告警并记录至预警记录页
15. 进入数据统计页，查看RSSI信号强度折线图、收发数据量统计及各解析字段历史趋势图
16. 断开蓝牙连接，返回设备扫描页
